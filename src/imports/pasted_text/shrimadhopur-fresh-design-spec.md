Design a modern mobile-first grocery delivery platform called "Shrimadhopur Fresh".

The platform provides local grocery delivery services only in Shrimadhopur, Rajasthan.

The design must include both:

1. Mobile app UI (Customer + Rider)
2. Modern interactive website landing page

The visual design should feel ultra modern, playful and premium.

---

DESIGN STYLE

Use a bright yellow themed design.

Primary colors:

#FFD84D (primary yellow)
#FFC107 (secondary yellow)
#FFF6CC (light background yellow)

Accent colors:

#1F2937 (dark text)
#34C759 (success)
#EF4444 (error)

Typography:

Inter / SF Pro

Use soft shadows, rounded cards, and minimal clean layouts.

---

3D ICON STYLE

Use soft 3D illustration icons similar to Guuulp style characters.

Examples:

• grocery bags
• vegetables
• delivery rider
• scooter
• shopping basket
• delivery box

Icons must look playful and modern.

---

INTERACTIVE WEBSITE (Landing Page)

Create a modern animated landing page.

The website should respond to:

• cursor movement
• scroll animations
• parallax effects
• floating 3D grocery icons
